# my-onchainkit-appfarcaster-mini
OnchainKit Farcaster Mini App تطبيق Web3 مبني باستخدام OnchainKit و Next.js يعمل على شبكة Base، ومُهيأ ليكون Mini App / Frame داخل Farcaster. يتيح للمستخدمين التفاعل onchain بسهولة (هوية، محفظة، معاملات) مع تجربة استخدام بسيطة وسريعة، ومناسب لبناء تطبيقات اجتماعية، توكنات، أو خدمات Web3 مدمجة مع Farcaster.
